# Car Showcase

Um site estático leve que exibe um carrossel de carros com navegação suave e design responsivo.

## Funcionalidades

- Slider de carros com controles anterior/próximo
- Layout de galeria responsivo
- Apresentação visual limpa para exibição de veículos
- Configuração simples com apenas HTML, CSS e JavaScript

## Estrutura do Projeto

- `index.html` - Marcações da página principal
- `style.css` - Regras de estilo e layout
- `cars.js` - Dados dos carros e lógica de geração de conteúdo
- `slider.js` - Navegação do slider e interatividade
- `img/` - Imagens usadas na vitrine de carros

## Como Executar

1. Abra `index.html` no navegador.
2. Use os controles do slider para navegar entre os carros.

## Personalizar

- Adicione ou atualize entradas de carros em `cars.js`
- Altere estilos em `style.css`
- Substitua imagens na pasta `img/`

## Observações

Este é um projeto estático, portanto não é necessário servidor. Ideal para demonstrações ou portfólios.
