# Slider de Exposição de Carros LEMOS

Um slider estático e elegante para exibição de carros, desenvolvido com HTML, CSS e JavaScript. Ele destaca cartões de veículos premium com transições animadas entre slides, reprodução automática, controles de navegação, indicadores de paginação e miniaturas de visualização.

## Recursos

* Layout de slider responsivo em tela cheia
* Transições suaves com efeito de fade entre os slides
* Reprodução automática com barra de progresso
* Botões de navegação anterior/próximo
* Indicadores de paginação e navegação por miniaturas
* Suporte para interação por teclado e ponteiro (mouse/toque)
* Marcação ARIA acessível para semântica de carrossel
* Destaques visuais personalizados e efeitos animados de revelação

## Arquivos

* `index.html` — estrutura principal da página e do slider
* `style.css` — estilos visuais, layout responsivo e animações
* `script.js` — comportamento do slider, reprodução automática, paginação e navegação
* `img/` — imagens utilizadas nos slides da apresentação

## Como Usar

1. Abra o arquivo `index.html` em seu navegador.
2. Utilize os botões esquerdo e direito para navegar entre os slides.
3. Clique nos indicadores de paginação ou nas miniaturas para ir diretamente a um slide específico.
4. Acompanhe a barra de progresso da reprodução automática, que avançará e trocará os slides automaticamente.

## Desenvolvimento

* Nenhuma ferramenta de compilação (build) é necessária.
* Edite diretamente os arquivos HTML, CSS ou JavaScript.

## Observações

* O projeto foi desenvolvido como uma demonstração estática e pode ser expandido com carregamento dinâmico de conteúdo ou cartões adicionais de veículos.
* O conteúdo textual está em português, mas possui uma estrutura compatível com o inglês para facilitar personalizações futuras.
